Nova versão Java 21

O Java 21 trouxe melhorias como padrões de registro, correspondência para switch e virtual threads com Project Loom, otimizando a concorrência. Além disso, aprimorou o GraalVM e introduziu novas APIs. Sendo uma versão LTS, garante suporte prolongado, tornando o Java ainda mais eficiente e moderno para desenvolvedores.
